# Recursos de IA Disponibles

Este documento detalla los recursos, herramientas y habilidades (skills) con las que cuenta el agente de IA para trabajar en el proyecto **Tienda JAFFS**.

## Servidores MCP (Model Context Protocol)

Los servidores MCP extienden las capacidades del agente conectándolo con herramientas y fuentes de datos externas.

*   **github-mcp-server:**
    *   **Descripción:** Proporciona herramientas para interactuar con la plataforma de GitHub.
    *   **Uso:** Permite realizar búsquedas de código, issues y pull requests, así como leer, crear y modificar archivos, ramas, pull requests y comentarios dentro de repositorios de GitHub. Ideal para gestionar el despliegue hacia Easypanel gestionando el control de versiones.

## Habilidades (Skills)

Las habilidades son instrucciones y flujos de trabajo especializados que mejoran la capacidad del agente para resolver tareas específicas.

*   **EasyPanel DevOps** (`easypanel-devops`):
    *   Experto en Dockerización, Docker Compose y despliegue en EasyPanel.
*   **TailwindCSS Development** (`tailwindcss-development`):
    *   Especialista en estilado de aplicaciones usando utilidades de Tailwind CSS v4. Útil para diseño responsivo, modo oscuro, layouts, colores, animaciones y garantizar la estética "premium" requerida.
*   **AI Context** (`ai-context`):
    *   Guía para crear, actualizar o mantener archivos de documentación de IA. Cubre estructura de archivos y convenciones.
*   **Cache Components** (`cache-components`):
    *   Proporciona patrones y mejores prácticas para Next.js Cache Components y Partial Prerendering (PPR).
*   **Flags** (`flags`):
    *   Ayuda para agregar o modificar feature flags experimentales de Next.js.
*   **Runtime Debug** (`runtime-debug`):
    *   Flujo de trabajo de depuración y verificación para regresiones en el bundle de ejecución y resolución de módulos (específico para proyectos Next.js o similares).
*   **Skill Lookup** (`skill-lookup`):
    *   Permite buscar, recuperar e instalar nuevas habilidades de agentes desde un registro utilizando herramientas MCP.
